Availability Feedback

The Mission

Hello, my name is Cody Carmony and for those of you who live in the tower what is one of the most annoying things you can think of? How about when you carry all of your laundry down to the laundry room only to discover that everything is full. I certainly have had this happen multiple times so what I created is a website where you could check to see which dryers or washers are open or how long until they are available. I am doing this as part of my final for Computing 230. This website will rely on you the users to accurately select which washer or dryer you are using and select the amount of time for which it will occupy. Once the time is up the website will then display what washer or dryer is available.


How it Works.

Go to the top right of this page and click on the availability tab which displays which washer and dryers are open. You will see three categories dryer, washer, dryer once you have the machine ID of the machine you want. The machine ID is displayed below the category name. Then go to the bottom of the page and click on the drop down tab to select your machine ID. Then do the same for your time, once you hit submit you should see that the machine you selected now displays the time until it is finished. If there are other times in your category and it is past that time once you submit your time it will automatically update and delete those times.
